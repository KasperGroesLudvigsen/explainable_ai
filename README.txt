The repository contains the deliverables in the module Explainable Artificial Intelligence (XAI) at City, University of London

The notebook called "shap" contains code that applies SHAP to explain the outputs of a CNN trained to classify MNIST digits. The notebook imports the modules cnn.py and get_data.py

Requirements: 
Shap must be installed. Run the command "pip install shap" in command prompt.
